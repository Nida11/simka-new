GetImageURL()

Description

Get the url of the new image resized from the image of a specified index in buffer.

Syntax

.GetImageURL(index, iWidth, iHeight);

Parameters

<int> index: specifies the index of image in buffer. The index is 0-based.
<int> iWidth: specifies the new width.
<int> iHeight: specifies the new height.

Return value

<string>

When an error occurs, check ErrorCode or ErrorString for error information.

Remarks

This method is valid only in HTML5 Edtion

The lowest limit for iWidth and iHeight is 150.
