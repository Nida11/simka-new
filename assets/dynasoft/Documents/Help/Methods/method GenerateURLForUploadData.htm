GenerateURLForUploadData()

Description

Generates a URL which will be used by the upload module to fetch the file/data to upload.

Availability

This API is added in v14.0 of the HTML5 edition and as of v14.0, it is only supported on Windows.

Syntax

.GenerateURLForUploadData(indices, enumImageType, asyncSuccessFunc, asyncFailureFunc);

Parameters

number indices Specifies the indices of images in buffer. The index is 0-based.

EnumDWT_ImageType enumImageType Specifies the format in which you'd like the images to be uploaded as.

<OnSuccess function> asyncSuccessFunc: callback function triggered when the operation succeeds.
<OnFailure function> asyncFailureFunc: callback function triggered when the operation fails.

Please refer to the function prototype OnSuccess or OnFailure .

Return value

string The URL of the file/data to be uploaded by the Upload Module.

Example

var dsUploadManager;
Dynamsoft.FileUploader.Init('', function(obj){dsUploadManager=obj}, function(){});
if(DWObject.HowManyImagesInBuffer<2){
    alert("Not enough images!");
}   
DWObject.GenerateURLForUploadData([0, 1], EnumDWT_ImageType.IT_PDF, function(res) {
	var serverUrl= "https://yourserver/{path}/youractionpage.aspx";
	var jobtemp = dsUploadManager.CreateJob();
	jobtemp.ServerUrl = serverUrl;
	jobtemp.SourceValue.Add(res, "uploadedFile.pdf");
	jobtemp.OnUploadTransferPercentage=function(job, percentage){console.log(percentage);}
	jobtemp.OnRunSuccess = function(job, responseString){console.log(responseString);};
	jobtemp.OnRunFailure = function(job, errorCode, errorString, responseString){ console.log(responseString)};
}, function(errorCode, errorString) {console.log(errorString);});
dsUploadManager.Run(jobtemp);
