IfAutoScroll

Description

Returns or sets whether to display the newly added image or keep the current one after an image(s) is imported into the Dynamic Web TWAIN viewer.

Data type

bool

Default value

true

If set to true, it will display the newly added image. If set to false, it will display the current one.

Syntax

.IfAutoScroll = true;

Remarks

This property is valid only in ActiveX (Windows) and HTML5 Edtion (Windows & Mac).

This property doesn't work when the view mode is set to -1 by -1

When an error occurs, check ErrorCode or ErrorString for error information.
